# Chua's circuit

Анимация по ссылке:
https://drive.google.com/file/d/1rlPJEHpjYkm0wG2Pe62WHmYmKeilx-U3/view?usp=sharing

Собрать проект можно двумя способами:
1) Через терминал (находиться нужно в основной папке проекта Chua_circuit):   
   cmake .  
   make  
   ./Chua_circuit
2) Или сразу  
   make  
   ./Chua_circuit
3) Через run в Clion
